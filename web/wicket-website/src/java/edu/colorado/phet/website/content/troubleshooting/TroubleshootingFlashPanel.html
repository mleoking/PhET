To run the Flash-based simulations you must have Macromedia Flash 8 (available free) or newer installed on your computer.

If you get a blank window when you try to launch a Flash simulation, you probably need a new version of the Flash player.

Why are right-to-left languages being displayed backwards?

For proper display of right-to-left languages (like Arabic and Hebrew) in Flash simulations, your language and region must be set to handle right-to-left languages.

For Windows XP: Select Start -> Control Panel -> Regional and Language Options. Under the Languages tab, make sure "Install files for complex script and right-to-left languages" is selected. Under the Advanced tab, select a language that supports right-to-left text under "Language for non-Unicode programs". Click Apply and restart your computer.
