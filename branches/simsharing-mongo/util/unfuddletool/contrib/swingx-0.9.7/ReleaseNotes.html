Release Notes

Release Date: 5 April 2009
This is the 0.9.7 development release

Project: SwingX

This release is due to last mnute issue found in the Calendar component found in just released SwingX 0.9.6.

JXMonthView

Fixed issues with loosing header color, refactored HeaderHandler and added tests. In addition also fixed problem with week counting being wrong for some locales.

RepaintManagerX

Ensured utilities always return correct translucent repaint manager.
