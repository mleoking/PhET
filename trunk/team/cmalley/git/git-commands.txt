Updating a submodule to HEAD:

cd common/phetcommon
git pull origin master
cd ../..
git submodule status
git add common/phetcommon
git commit –m “Updated phetcommon to HEAD”
git push origin master


-----

Add a submodule dependency:

cd some_dir
git submodule add git://github.com/phetsims/myrepo.git myrepo
git commit -m "add submodule myrepo"
git push origin master

-----

Update all submodules to HEAD so that changes can be committed: (UNTESTED!)

git submodule foreach --recursive 'git pull origin master && git checkout master'
