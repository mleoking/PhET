This directory contains the software for PhET simulations.

Running the build script (build.bat for Windows, build.sh for UNIX-based systems)
will display a graphics user interface to assist you in building and running
PhET simulations.

Please see LICENSE.txt and phet-license.txt for PhET's licensing policy.
Third-party libraries and tools (located in the "contrib" directory)
have different licensing policies. Please see each contrib subdirectory
for more information.
