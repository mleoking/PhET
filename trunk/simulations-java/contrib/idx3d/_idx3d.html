idx3d is a quite powerful realtime 3d engine written entirely in Java (1.1).
Copyright of all parts of the idx3d engine is 1999/2000 by Peter Walser.
Feel free to use the engine or source code for your non-commercial projects. If you want to use it for commercial purpose, please contact me with e-mail.
The current version is 3.0 (a.k.a. idx3dIII).
Note: The documentation is now almost done, same for the API docs. :)

And now something for your eyes :)
Click on any thumbnail to run the corresponding demo...

Demo 1: Lathe Objects

A wineglass and a torus created as lathe objects with idx3d_ObjectFactory.

Demo 2: 3d Field

A phong shaded textured 3d surface created from a data set with idx3d_ObjectFactory.

Demo 3: Stones

Phong shaded textured stones, imported from a 3ds file.

Demo 4: Torusknot

The torus knot with p=5 and q=1 has been created with idx3d_ObjectFactory. You can switch between materials at runtime.

Demo 5: Materials

Various materials used in the same scene with fancy moving objects. Imported from a 3ds file.

Demo 6: Lensflare

Stones from demo3, this time they're metallic. Demo for the LensFlare plugin.

Demo 7: Venus in Cave

Venus of Milo in Cave with lensflare. Looks damned pretty...

Demo 8: Chrome Mech

A metallic mech warrior. Still wondering why envmapping is so stunning....

Demo 9: Blueprint

A transparent blue mech warrior with reflections.

Demo 10: Linkable

Demo for the idBuffer. The objects are associated with URLs, so you can open a site by clicking at them. =)

Demo 11: Evil Seed

A nice demon head I created while learning 3d modelling, with an evil burning inside its brain.
