idx3d Material Lab

The idx3d Material Lab helps you creating materials. You can customize the object's color, transparency and reflectivity, and assign a texture and an environment (reflection) map to it. The texture and envmap can either be imported from JPG,GIF and PNG files or be created with the idx3d_TextureGenerator. The in-built Texture Lab lets you play with the various parameters of the texture generators, and only its parameters will be saved into the material file, so the texture is generated on-the-fly when needed, letting you save time as no bitmap data has to be downloaded to the customer.

The Material Lab is an application and is contained in the idx3d.zip archive you can download.
You can run it with java idx3dMaterialLab.
