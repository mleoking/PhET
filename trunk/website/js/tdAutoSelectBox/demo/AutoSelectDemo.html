Auto SelectBox Examples

Basic Example

The following examples should illustrate the most basic features of tdAutoSelectBox. Just start typing the name of a state, or select it from the list.

Select or type in a State.

As you can see, there really is nothing to it. It is simple and intuitive to use. This is the way select boxes should be!

With Freetype Enabled

This example is similar to the previous, but has Freetype enabled. Freetype allows as user to select from the list as above, but he can also type any value he wishes. This is less secure (as it allows a user to submit any kind of value) but may have its uses on certain kinds of web forms.

Select or type an item

Javascript Alert Example

This example is similar to the basic example, but has the added power of executing a pop-up alert when the user clicks on an item in the list. Otherwise, if he just types the item, he should click "check" to check what he has typed. This select box also has a default value set to "Horse". The autotruncate feature is also switched "off" so that the list will not grow shorter or change as you type.

Select or type the name of an animal.


Any kind of javascript function can be invoked in this way, including code that auto-submits the form. However, this is for more advanced programs, as you will need to know how to program the new javascript functions yourself.

Multi-SelectBox Example

Here is an example of a select box, which feeds a multi-select area. Simply select an item from the list, and click "add". If you want to remove an item from the multi-list, click it (so that it turns gray with a line struck through) and click "del".

Add one or more items
(Click an item and select "Del" to remove)

You may notice that you cannot add more than one occurrence of the same item to the list. Behind the scenes, the multi-select area is really a list of checkboxes (you just can't see the checkboxes), and behaves exactly like an HTML multi-selectbox or set of checkboxes.

Multi-SelectBox with Auto-Add

This is the same as above, but has Javascript added such that an item is automatically added when it is selected from the drop-down list.
