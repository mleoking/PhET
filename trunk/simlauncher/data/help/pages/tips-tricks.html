Tips & Tricks

Hiding and showing thumbnail graphics

You can hide or show the simulations' thumbnails graphics on either the My Sims or Add/Remove/Update Sims pane by clearing or checking the Show Tumbnails checkbox at the bottom right of the window.

Speeding sp simLauncher startup

When SimLauncher starts, it tries to connect with the PhET server to see if there are any updates available. This can be a time-consuming task in some environments. You can speed up the the startup process by turning this feature off. To do this, clear the Automatically check for updates on startup checkbox in the Options menu.

Prevent students from installing and removing simulations

You can allow or disallow certain users, such as students, from installing or removing simulations by controlling their accounts' access to the directory in which SimLauncher stores simulations. The default directory is C:\Program Files\PhET SimLauncher\localRoot on a PC. On a Mac or Linux computer, it is /PhET SimLauncher/localRoot. If you have installed SimLauncher somewhere other than the default directory, the simulations will be in the localRoot directory under the directory you specified.

If you want to prevent certain users from being able to install or remove simulations, deny them write priviledges to that directory.

If all else fails...

If you can't get your simulations to launch or SimLauncher is not behaving properly, you can reset the program by choosing "Clear cache" from the File menu. Be advised, however, that this action will uninstall any simulations you have installed.
