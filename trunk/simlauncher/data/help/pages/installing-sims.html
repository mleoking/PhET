Installing Simulations

Installing a Single Simulation

Open the PhET SimLauncher. Make sure you are in the Add/Remove/Update Sims tab, where there will be a list of all the sims, and it will show which simulations are already installed on the computer, as well as if updates are available for any of the simulations. To download a simulation that is not already on the computer, just check the box next to the simulation you want to download, and click the Install/Update button under the list of simulations.

Installing All the Simulations of a Particular Category

On the left side of the screen, there is a list of categories. If you want to view all the simulations in a particular category, select the category, and the list of simulations will come up for that category. Then you can select which simulations from that category you would like to install. To install all of the sims from a particular category, select the category, and then click on Select All at the bottom. Then click on Install/Update to install or update the sims.

Installing All Simulations

To install all of the sims available, make sure that All simulations is selected in the list of categories on the left, and then click on Select All at the bottom. Then click on Install/Update to install the sims.
