Running Simulations

  • Go to My Sims.
  • Select the category that has the simulation you wish to run.
  • To launch a simulation, do one of the following:
    • Double-click on the simulation.
    • Select the simulation, then select Simulation >Launch Simulation from the main menu.
    • Select the simulation, then click the "Launch" button.
    • Right-click on the simulation and select "Launch" from the pop-up menu.
