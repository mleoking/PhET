Uninstalling Simulations

Uninstalling a Single Simulation

Go to the Add/Remove/Update Sims tab. Select the simulation you want to uninstall (it will be highlighted in blue when it is selected). In the “Simulation” pull-down menu at the top of the screen, select Remove, or right-click with the mouse and select Remove from the pop-up menu.

Uninstalling All the Simulations in a Category

Go to the Add/Remove/Update Sims tab. In the Categories area on the left side, select the topic for which you would like to uninstall sims. At the bottom of the screen, click on the Select All button. Make sure that all simulations are selected with a checkmarks in the small boxes just to the left of the list of simulations. Then, from the Simulations pulldown menu at the top of the screen, select Remove, or right-click with the mouse and select Remove from the pop-up menu.

Uninstalling Several Selected Simulations

Go to the Add/Remove/Update Sims tab. If the simulations you wish to uninstall are all in the same category, you can cot to that category. Otherwise, go select All simulations in the Categories list. Select each of the simulations you wish to uninstall. Then, from the Simulations pulldown menu at the top of the screen, select Remove, or right-click with the mouse and select Remove from the pop-up menu.
